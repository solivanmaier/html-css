Texto Marcado

Podemos criar também um texto marcado usando usando a tag MARK.

Para sublinhar use a tag INS (semântica)

Não use a tag U (não semântica)

Quando você precisar de um texto ou número pequeno em cima use a tag SUP

Quando quiser um texto abaixo EX: H20 use a tag SUB
